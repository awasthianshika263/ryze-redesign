# Ryze.ai Website Redesign

**Live Demo:** https://ryzeai.netlify.app/

**GitHub Repository:** (https://github.com/awasthianshika263/ryze-redesign)

## Overview
Redesigned the Ryze.ai homepage and two inner pages (Features & Pricing) to make it modern, clean, professional, and user-friendly.

## Key Improvements
- **Visual Appeal:** Blue-gray tech color palette, subtle hero background, card-based layout with icons instead of tables.
- **Clarity:** Clear value proposition in hero ("AI manages ads automatically, boosts ROAS"). Added transparent Pricing page and "Wall of Love" testimonials with real client quotes.
- **User Experience:** Prominent CTAs ("Book a Demo", "Get Started"), smooth scrolling, hover effects, fully mobile-responsive.
- **Modular Structure:** Shared header/footer, reusable card components – easy to update.
- **Extra Features:** Icons in features cards, JavaScript for interactivity (smooth scroll, hover lift, button alerts).

Built with HTML, CSS, and minimal JavaScript – fast, lightweight, and maintainable.
